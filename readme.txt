== Description ==

Display an overview of Wordpress galleries used on child pages.

== Installation ==

1. Upload the directory `adm-gallery` to the `/wp-content/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `[adm-gallery]` on the page where you want to show the gallery overview
